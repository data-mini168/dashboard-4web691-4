☁️ ตั้งค่า Cloud Sync (Firebase)

เชื่อมต่อ Firebase เพื่อให้ข้อมูลซิงค์ข้ามทุกบราวเซอร์และทุกเครื่องแบบ Realtime

วิธีสร้าง Firebase ฟรี (5 นาที)
  1. ไปที่ console.firebase.google.com
  2. กด Add project → ตั้งชื่อ → Create
  3. เลือก Build → Realtime Database → Create database
  4. เลือก Start in test mode → Enable
  5. ไปที่ Project Settings ⚙️ → General
  6. กด </> Add app → ลงทะเบียน
  7. Copy firebaseConfig มาใส่ด้านล่างนี้
✅ เชื่อมต่อแล้ว
⚠️ หมายเหตุ
• Firebase Realtime DB ฟรี 1GB ข้อมูล / 10GB transfer/เดือน
• ข้อมูลจะซิงค์ทุกบราวเซอร์ทันทีเมื่อกด บันทึก
• ควรตั้ง Security Rules ก่อน deploy จริง
